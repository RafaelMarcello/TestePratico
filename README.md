## Install

```sh
composer install
```

## Usage

```sh
cp .env.example .env
php artisan key:generate
Add your database config in the .env file (you can check my articles on how to achieve that)
php artisan migrate:fresh --seed
```

## Run tests

```sh
php artisan serve
```
